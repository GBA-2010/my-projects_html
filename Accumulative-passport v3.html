ПАСПОРТ ПРОЕКТА "Sanya Ultimate Travel Guide"

Название: Интерактивный мобильный гид по Санья

Версия: 3.3 (Final Release Candidate)

Дата обновления: 11.01.2026

Разработчик: Gemini AI

1. ЦЕЛЬ И НАЗНАЧЕНИЕ

Создание одностраничного веб-приложения (SPA) для туристов в г. Санья (о. Хайнань), работающего в офлайн-режиме (при кэшировании) и предоставляющего удобную навигацию, справочник мест и переводчик фраз.

2. ТЕХНОЛОГИЧЕСКИЙ СТЕК

  • HTML5: Структура и семантика.
  • CSS3: Адаптивный дизайн, Flexbox/Grid, Glassmorphism, анимации.
  • JavaScript (ES6+): Логика приложения (без фреймворков).
  • Leaflet.js (v1.9.4): Картографический движок.
  • Leaflet Routing Machine: Построение маршрутов (OSRM).
  • FontAwesome (v6.4.0): Векторные иконки.
  • Web Speech API: Синтез речи (TTS) для озвучивания китайских фраз.

3. ФУНКЦИОНАЛЬНЫЕ МОДУЛИ

3.1. Модуль "Карта и Навигация"

  • Google Maps Style: Дизайн карты, привычный пользователям.
  • Жесткое правило Геолокации (Hard Rule v4):
    • Приоритет: Реальный GPS.
    • Защита: Если браузер/контекст блокирует GPS — мгновенное переключение на Демо-режим (Центр Санья) без ошибок.
    • Ручной режим: Возможность указать точку "Я" кликом по карте.
  • Умное меню: Контекстное меню при клике по карте ("Поехать отсюда" / "Поехать сюда").
  • Маршрутизация: Построение маршрутов (Авто/Пешком) внутри приложения.
  • HUD: Панель активной навигации с временем и дистанцией.

3.2. Модуль "Поиск и Фильтры"

  • Глобальный поиск: Интегрированная в шапку панель поиска с автодополнением по базе мест.
  • Быстрые фильтры: Чипсы (Отели, Еда, Пляжи, ТЦ) для быстрой фильтрации меток на карте.

3.3. Модуль "Объекты (POI)"

  • База данных: Включает ключевые отели (Atlantis, MGM, Ritz), ТЦ (Ананас, Joy City) и достопримечательности.
  • Карточка объекта: Выезжающая шторка с информацией и действиями ("Такси", "Маршрут").
  • Такси-помощник: Показывает карточку с крупным китайским текстом назначения.

3.4. Модуль "Разговорник и Еда"

  • Фразы: Категоризированный список с озвучкой.
  • Меню: Карточки популярных блюд с фото и кнопкой заказа на китайском.

4. ИНТЕРФЕЙС (UI/UX)

  • Mobile-First: Интерфейс оптимизирован под управление пальцем (крупные кнопки, нижняя навигация).
  • Z-Index Management: Исправлены наложения окон (поиск, шторки, тосты не перекрывают друг друга).
  • Обратная связь: Toast-уведомления вместо системных alert-окон.

5. ИСТОРИЯ ВЕРСИЙ

  • v1.0: Базовая версия (карта + фразы).
  • v2.0: Редизайн под Google Maps, плавающие фильтры.
  • v2.5: Внедрение таймера GPS для мобильных.
  • v3.0: Умное меню клика, поиск мест.
  • v3.3: Перенос поиска в шапку, финальная отладка слоев и геолокации.
